This folder should contain PNG icons for the PWA.
You can generate them using the generate-icons.html file or create them manually.

Required icon sizes:
- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

For now, the PWA will work without icons but will show default browser icons.
